# AGENT.md

This includes guidance when working with code in this repository.

## Overview

This contains curriculum materials for coaches running a "Vibe Coding Happy Hour" for K12 students (ages 8–14).

## Token Efficiency

Use caveman prompts when communicating: drop filler words, articles, and pleasantries. Say "add timer segment 3" not "Could you please add a countdown timer to segment 3?". This keeps context lean and responses faster.
