# AGENT.md

This includes guidance when working with code in this repository.

## Code Principles

1. **Think Before Coding**: Don't assume. Don't hide confusion. Surface tradeoffs.
2. **Simplicity & Lean Always**: Minimum code that solves the problem. No redundancy.
3. **Surgical Changes**: Touch only what you must. Clean up only your own mess.
4. **Goal-Driven Execution**: Define Success Criteria. Loop Until Verified.
5. **Token Efficiency**: Use caveman prompts when communicating: drop filler words, articles, and pleasantries.
