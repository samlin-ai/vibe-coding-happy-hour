# Vibe Coding Happy Hour
This is a course for coaches to introduce Vibe Coding to K12 students.

## 2 hrs course

1. 30 min. Introduction to AI & Gemini CLI for all on Google Meet 
2. 60 min. Breakout Session: Vibe Coding Your First Game
   a. 3-5 studens with 1 coach in person or virtually in a breakout room
   b. The coach guide the team to build their first game according to [the instructions](./coach-guide-gemini-cli.md). 
3. 20 min. Show & Tell for each team for all on Google Meet
4. 10 min. Closing
